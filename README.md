# 📊 Proyecto Big Data - D1 (Spark & Kafka)

Análisis de datos de ventas simuladas de Tiendas D1 utilizando **Apache Spark** para procesamiento en batch y **Apache Kafka** para procesamiento en tiempo real (streaming).

---

## 🎯 Objetivo

Desarrollar una solución de Big Data que permita analizar datos de ventas, identificar patrones de consumo y procesar información tanto histórica como en tiempo real.

---

## 🛠️ Tecnologías

* Python
* Apache Spark
* Apache Kafka
* PySpark
* Matplotlib

---

## 📂 Estructura del proyecto

```
proyecto-bigdata-d1-spark-kafka/
│
├── batch.py
├── kafka_producer.py
├── streaming.py
├── data/
│   └── superstore.csv
└── README.md
```

---

## ▶️ Ejecución

Instalar dependencias:

```
pip install pyspark kafka-python matplotlib
```

Ejecutar:

```
python batch.py
python kafka_producer.py
python streaming.py
```

## 📈 Resultados

* Análisis de ventas por categoría
